Demo Wrap Up

So we've now finished the Riot API demo! If you followed everything closely (or just copied and pasted my code, which is fine), then you should now have a working application that uses the Riot API! From here you can add more constants to RiotConstants.py to expand what APIs and regions you use, add wrapper functions to RiotAPI.py to request more information from the Riot API, and add more reply variables to Main.py to manipulate the data you want even further. As promised, below is the code for the Main.py file that is cleaned up a bit more, including only one print statement that shows your final result: The win/loss ratio of any summoner name you enter!



I hope this demo helped build a foundation for you to manipulate the Riot API however you'd like. The amount of raw information you can get using the Riot API is astounding, so let the creative juices flow! Don't stop at just finding a number or two, just keep building and building to try and build something big out of all this information. I'm personally in the midst of making my own personal project which involves most of this demo being used as its foundation. You can learn more about this project by checking out my projects page in the near future. All the code used in this demo is available on my github page. I'll likely update some of the files with further code (more work in the constants file, and their respective functions defined in RiotAPI) once my project is finished, so keep an eye out for that as well.


If anything in this demo was unclear, I'd be happy to clarify any part of the process displayed. Just leave me a comment below or contact me by email and I'll be happy to help.

Thanks for reading, and enjoy!
